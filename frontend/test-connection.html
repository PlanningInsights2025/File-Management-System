ðŸ“‹ Setup Checklist

            

                Step 1: Install backend dependencies
                
cd backend
npm install

            

            

                Step 2: Create .env file in backend folder
                
PORT=5000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=fms_db
JWT_SECRET=your-secret-key

            

            

                Step 3: Setup database
                
CREATE DATABASE fms_db;
mysql -u root -p fms_db < database/users_table.sql

            

            

                Step 4: Start backend server
                
cd backend
npm start

            

            

                Step 5: Open this test page and click "Test Backend Connection"
